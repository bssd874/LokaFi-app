# LokaFi Web

React + Vite frontend for LokaFi. See the root [README](../README.md) for full setup, Freighter Testnet setup, environment variables, and demo flow.

Common commands:

```bash
npm install
copy .env.example .env
npm run dev
npm run lint
npm run build
```

Open the local frontend at `https://localhost:5173` so Freighter has a secure browser context.
