# LokaFi API

Laravel REST API for LokaFi. See the root [README](../README.md) for full setup, PostgreSQL configuration, migrations, seed data, tests, API summary, and demo flow.

Common commands:

```bash
composer install
copy .env.example .env
php artisan key:generate
php artisan migrate
php artisan db:seed --class=LokaFiDemoSeeder
php artisan test
php artisan serve
```
